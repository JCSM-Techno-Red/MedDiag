# 🩺 Sistema de Diagnóstico Médico

Sistema desktop para auxílio diagnóstico baseado em sintomas, com cadastro de pacientes, histórico e exportação de relatórios.

## Funcionalidades
- Cadastro completo de pacientes
- Seleção de sintomas (base com +500 itens)
- Motor de diagnóstico com pesos e porcentagem
- Histórico e exportação de relatórios

## Como executar
1. Tenha Python 3.10+ instalado
2. Execute `python main.py`

## Tecnologias
- Python 3
- Tkinter
- JSON
